## Features

- Light/dark mode toggle
- Minimal desgin
- Customizable


## Sections
- About
- Projects
- Experience
- Skills
- Contact

## Installation

Install react-personal-portfolio with npm


```
  git clone https://github.com/shehabshalan/react-personal-portfolio
  cd react-personal-portfolio
```
```
  npm run dev
```

## Tech Stack
HTML, CSS, React, MUI, mui-icons
